# Project Status

Dit bestand is bedoeld als overdracht naar een nieuwe chat of een andere computer.

## Waar staat het project nu

Het project is een eerste werkende basis van een webapp voor een transportplanner.

De app is gebouwd met:

- `React`
- `Vite`

De code staat op GitHub:

- `https://github.com/RobMaas01/Transportplanner`

## Wat al klaar is

- er is een werkend `Vite + React` project
- de hoofdapp staat in `src/App.jsx`
- de app is gebaseerd op een eerder voorbeeldbestand
- dat oude voorbeeldbestand is verwijderd
- opslag werkt nu via `localStorage`
- de repository staat op GitHub
- de branch is `main`

## Belangrijke keuze

De bedoeling is om op de andere computer vooral te werken met:

- `Codex`
- `ChatGPT`

En zo min mogelijk direct met:

- terminalcommando's
- ingewikkelde handmatige setup

## Waarom dat zo is

De app is bedoeld voor verdere ontwikkeling op een andere computer.

Daarom is de documentatie zo geschreven dat iemand:

- rustig kan instappen
- niet alles technisch hoeft te weten
- vooral met hulp van `Codex/ChatGPT` verder kan

## Wat bewust niet meer leidend is

- `bibliotheek_transport.tsx` is verwijderd
- `window.storage` wordt niet meer gebruikt
- `Docker` is niet meer onderdeel van de hoofdroute in de documentatie

## Wat nog niet klaar is

- verdere ontwikkeling van de app
- koppeling met `Supabase`
- koppeling met `Vercel`

## Volgende beste stap

De volgende beste stap op een andere computer is:

1. project openen
2. met `Codex/ChatGPT` laten helpen bij opstarten
3. app lokaal laten draaien
4. daarna pas verder bouwen aan de app
