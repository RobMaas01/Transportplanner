# Start Hier

## Uitleg

Dit project is een eerste basis van een webapp voor een transportplanner.

De app is gemaakt met:

- `React`
- `Vite`
- `GitHub`

Later kunnen daar nog bij komen:

- `Supabase`
- `Vercel`

Op dit moment is het belangrijkste doel:

- het project openen op een andere computer
- lokaal starten
- rustig verder ontwikkelen

De voorkeur is:

- zoveel mogelijk werken met `Codex/ChatGPT`
- zo min mogelijk zelf in de terminal doen

Je hoeft dus nog niet alles tegelijk te begrijpen.

De eerste stap is alleen:

- de juiste programma's installeren
- het project ophalen
- de app lokaal starten

Dit project staat op GitHub:

- `https://github.com/RobMaas01/Transportplanner`

## Wat eerst nodig is

Installeer op de computer:

- `VS Code`
- `Node.js`
- `Git`

### Hoe installeren op Windows

#### VS Code

1. Ga naar:
   - `https://code.visualstudio.com/download`
2. Kies:
   - `User Installer x64`
3. Installeer het programma met de standaardopties.

#### Node.js

1. Ga naar:
   - `https://nodejs.org/en/download`
2. Kies de `LTS` versie.
3. Installeer met de standaardopties.
4. Open daarna `VS Code`.
5. Open de projectmap als die al op de computer staat, of gewoon een lege map.
6. Open onderin een opdrachtvenster via:
   - `Terminal`
   - `New Terminal`
7. Dat venster heet de terminal.
   Daar kun je commando's typen.
8. Controleer daarna in dat venster:

```bash
node -v
npm -v
```

Voor nu zijn deze drie genoeg:

- `VS Code`
- `Node.js`
- `Git`
